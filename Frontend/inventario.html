Importar desde Excel

Ideal para cargar o actualizar muchos productos a la vez. Si no manejas muy bien Excel, no te preocupes: sigue los pasos.

  1. Descarga la plantilla base.
  2. Abre el archivo y llena las columnas. Las obligatorias son SKU y Nombre.
  3. Guarda los cambios y vuelve aquí para subir el archivo.
Consejos:
  • En Original (SI/NO) puedes escribir SI o NO.
  • Para decimales, puedes usar coma o punto. Ej: 85.000 o 85,000.
  • Si un SKU ya existe, se actualiza (precio, stock, etc.). Si no existe, se crea.
